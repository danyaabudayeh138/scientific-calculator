# 🧮 Scientific Calculator

## 📌 Overview
This is a **Scientific Calculator Web Application** built using HTML, CSS, and JavaScript.  
It performs basic and advanced mathematical operations with a modern and responsive UI.

---

## ✨ Features
- Basic operations (+, -, ×, ÷)
- Scientific functions:
  - Square root (√)
  - Power (^)
  - sin / cos / tan
  - logarithm (log)
  - percentage (%)
  - constant π
- Clean and modern UI
- Responsive design
- Easy to use interface

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

-

## 🚀 How to Run
1. Download or clone the repository
2. Open `calculator.html`
3. Run it in any browser
